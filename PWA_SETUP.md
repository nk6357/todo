## 🔧 Что нужно сделать

### 1️⃣ Добавьте иконки приложения

Поместите в папку `icons/`:
- `icon-192x192.png` - 192х192 пикселей
- `icon-512x512.png` - 512х512 пикселей  
- `icon-1024x1024.png` - 1024х1024 пикселей (опционально)

**Как создать иконки:**

#### Вариант 1: Использовать Real Favicon Generator
1. Откройте https://realfavicongenerator.net
2. Загрузите свое изображение (минимум 560x560 px)
3. Нажмите "Generate your Favicons and HTML code"
4. Skачайте favicon package
5. Распакуйте PNG файлы в папку `icons/` этого проекта

#### Вариант 2: Использовать ImageMagick (если установлен)
```bash
cd icons/
convert original.png -resize 1024x1024 icon-1024x1024.png
convert original.png -resize 512x512 icon-512x512.png
convert original.png -resize 192x192 icon-192x192.png
```

#### Вариант 3: Использовать Paint/Photoshop/GIMP
Просто создайте три изображения размерами 192x192, 512x512 и 1024x1024 пикселей

### 2️⃣ Загрузите на сервер (HTTPS обязателен!)

Убедитесь, что приложение доступно по **HTTPS**:
- GitHub Pages - автоматически HTTPS ✅
- Netlify - автоматически HTTPS ✅
- Vercel - автоматически HTTPS ✅
- Собственный сервер - настройте SSL сертификат

### 3️⃣ Установите приложение на iPhone

#### Способ установки:

1. **Откройте Safari на iPhone** (не Chrome, не Firefox!)
2. **Перейдите на адрес:** `https://your-domain.com`
3. **Нажмите кнопку "Поделиться"** (квадрат со стрелкой снизу)
4. **Прокрутите вниз и выберите "На экран Домой"** (Add to Home Screen)
5. **Нажмите кнопку "Добавить"** в правом верхнем углу
6. **Готово!** Приложение появится на главном экране

#### Тестирование офлайн-режима:

1. Откройте приложение через иконку на главном экране
2. **Включите Авиарежим** (Настройки → Авиарежим)
3. Приложение должно работать нормально!

## ⚠️ Частые проблемы и решения

| Проблема | Решение |
|----------|---------|
| Не работает "На экран Домой" | 1. Используйте Safari (не Chrome) 2. Проверьте HTTPS 3. Откройте сайт напрямую |
| Service Worker не регистрируется | 1. Очистите кэш Safari: Настройки → Safari → Очистить историю 2. Проверьте формат manifest.json |
| Данные теряются при перезагруске | iOS может очищать хранилище при нехватке места. Это нормально - просто переустановите приложение |
| Не видны иконки | Убедитесь, что PNG файлы находятся в папке `icons/` с правильными именами |
| Черный экран при запуске | 1. Перезагрузите приложение (закройте и откройте) 2. Очистите кэш Safari |

## 🔐 Безопасность данных

Все данные сохраняются **локально на устройстве** в `localStorage`:
- Результаты пробных тестов
- Выбранная тема приложения
- История посещений

**Данные НЕ передаются на серверы!**

## 🔄 Обновление приложения

Когда вы обновите файлы на сервере:
1. Приложение автоматически проверит обновления каждые 6 часов
2. Если доступно обновление, появится диалог
3. Нажмите "Установить" и приложение перезагрузится с новой версией
